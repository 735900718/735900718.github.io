CSS渐变边框

2017-04-19 2 min read # css

渐变？！不就是linear-gradient么！这个简单啊，这样那样一下不就出来了么。。。然后就出错了= =

先来复习一下linear-gradient()这个CSS函数：

第一个参数是angle渐变方向：可以不写，默认为to bottom ，也可以写作: 180deg。

可选值有：
to left：设置渐变为从右到左。相当于: 270deg。
to right：设置渐变从左到右。相当于: 90deg。
to top：设置渐变从下到上。相当于: 0deg。
to bottom：设置渐变从上到下。相当于: 180deg。

后面的参数是颜色参数 同时可以标记颜色坐标，以百分比的形式标注，中间以空格隔开。

例：linear-gradient(to bottom, #fff 0%, #333 100%);

好了，回忆结束。

border使用渐变的方法：

border-image: linear-gradient(to bottom, #fff 0%, blue 100%);

设置完之后绚丽的色彩渐变就出来了~~

但是当加上圆角border-radius的时候问题就出来了，两个属性不能同时使用，这就很尴尬了。。。

找了一些资料后，换一个思路：

使用背景色渐变，然后在再上面盖一层模块，给下层背景空留出一个边框的距离，于是一个渐变的边框出来~ 下面是代码：

	<style lang="less">
	@besic: #0079F3;
	@border-width: 3px;
		.btn{
			display: inline-block;
		  	position: relative;
		  	margin: @border-width;
		  	width: 10rem;
			line-height: 2.75rem;
			height: 2.75rem;
			text-align: center;
			color: @besic;
		    border-radius: 16px;
		    background: #fff;
		    cursor: pointer;
		    transition: all .5s;
		    &:after {
		    	position: absolute;
			    top: -@border-width; bottom: -@border-width;
			    left: -@border-width; right: -@border-width;
			    background: linear-gradient(to right,#4AE6B1, #4AA2F4);
			    content: '';
			    z-index: -1;
			    border-radius: 16px;
		    }
		    &:hover{
		    	color: #fff;
		    	background: linear-gradient(to right,#4AE6B1, #4AA2F4);
		    }
		}
	</style>
